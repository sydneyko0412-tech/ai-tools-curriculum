# My Prompt Library
 
이 문서는 내가 실무에서 자주 쓰는 프롬프트를 모아두는 라이브러리입니다.  
툴별·목적별로 섹션을 나누고, 마음에 드는 프롬프트를 그대로 복붙해서 추가해 사용하세요.
 
***
 
## 1. Email / Communication
 
### 1-1. 상사/리더에게 주간 보고 메일 작성
 
**설명**  
주간 성과·이슈·다음 주 계획을 정리할 때 사용하는 프롬프트입니다.
 
**Prompt (모델에 그대로 붙여넣기)**
 
역할: 한국 FMCG 회사의 브랜드/데이터 마케터, 4년차  
 
상황:  
- 나는 이번 주에 진행한 업무들을 bullet로 정리해 둘 거야.  
- 이 내용을 바탕으로, 상사에게 보낼 주간 보고 메일을 작성해줘.  
 
요청사항:  
1) 메일 제목 제안 3가지  
2) 본문 구조: 이번 주 요약 → 주요 성과 → 이슈/리스크 → 다음 주 계획  
3) 각 섹션은 2~3줄 내로 간결하게 작성  
4) 톤은 공손하지만, 불필요하게 길지 않게  
 
아래에 이번 주 업무 내용을 bullet로 줄게.
 
[여기에 이번 주 업무 bullet들을 붙여 넣기]

## 2. Report / Analysis
 
### 2-1. MMM/ROI 결과 요약용 프롬프트
 
**설명**  
MMM/ROI 분석 결과를 임원용 한 페이지로 요약할 때 사용하는 프롬프트입니다.
 
**Prompt**
 
역할: 디지털 마케팅 데이터 애널리스트  
 
목표:  
- 아래에 붙여넣은 텍스트(또는 표)는 MMM/ROI 분석 결과 요약이야.  
- 이 내용을 기반으로 임원용 한 페이지 요약을 만들고 싶어.  
 
요청사항:  
1) Executive Summary (3~5줄)  
2) 핵심 인사이트 5개 (각 1~2줄, 채널/크리에이티브/프로모션 관점)  
3) Recommended Actions 5개 (각 1줄, 실행 가능하게)  
 
제약:  
- 숫자는 최대한 그대로 유지  
- 가설/추측이 필요한 부분은 “추정”이라고 명시  
 
아래에 분석 결과 요약을 붙일게.
 
[여기에 MMM/ROI 결과 텍스트 또는 표 요약을 붙여 넣기]

## 3. Campaign / Ideation
 
### 3-1. 시즌 캠페인 아이디어 발산
 
**설명**  
여름/겨울 등 시즌 캠페인 아이디어를 뽑을 때 사용하는 프롬프트입니다.
 
**Prompt**
 
역할: 한국 FMCG 브랜드 마케터  
 
브리프:  
- 브랜드: [브랜드명]  
- 카테고리: [카테고리명 예: 보디오일]  
- 시즌: [예: 2026년 여름]  
- 타겟: [예: 20~30대 여성, 건성 피부, 온라인 구매 중심]  
- 주요 채널: 인스타그램, 유튜브, 카카오 선물하기  
 
요청사항:  
1) 시즌 캠페인 컨셉 10개 (이름 + 2줄 설명)  
2) 각 컨셉별로 가장 적합한 메인 채널 1개  
3) 각 컨셉별로 측정할 핵심 KPI 1개  
 
출력 형식:  
- 표 형식: 컨셉 이름 / 설명 / 메인 채널 / 핵심 KPI  
 
추가:  
- 너무 전형적인 “할인/세일” 컨셉은 피하고, 브랜드 자산을 강화할 수 있는 방향으로 제안해줘.
 
***
 
## 4. Meeting / Collaboration
 
### 4-1. 회의록 정리 & 액션 아이템 추출
 
**설명**  
회의 끝나고 자유로운 메모를 구조화할 때 사용하는 프롬프트입니다.
 
**Prompt**
 
역할: 프로젝트 매니저  
 
상황:  
- 아래에 회의 중에 적어둔 메모를 그대로 붙일 거야.  
- 키워드 위주로 섞여 있을 수 있어.  
 
요청사항:  
1) 회의 목적 1~2줄 요약  
2) 주요 논의 내용 3~5줄 요약  
3) 표 형식으로 정리:  
   - 항목: 액션아이템 / 담당자 / 데드라인 / 관련 메모  
4) 빠진 정보(담당자/데드라인 등)가 있으면 “추가 필요”라고 표시  
 
아래에 회의 메모를 붙일게.
 
[여기에 회의 메모를 붙여 넣기]
