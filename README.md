### Week 1 – 멀티툴 지도 & 역할 정의

**주간 목표**

- Copilot / Gemini / Claude / Perplexity 각각의 역할을 명확히 정의.
- 같은 요청을 4개 툴에 던져보고 차이를 체감.

#### Day 1 – 툴 전반 훑기 + 1줄 정의

- 학습 (20분)
  - Perplexity at Work Guide 개요 읽기  
    - https://www.perplexity.ai/enterprise/perplexity-at-work
  - LinkedIn Learning – Empower Your Productivity with Google Gemini 코스 소개 영상 1개 시청  
    - https://www.linkedin.com/learning/empower-your-productivity-with-google-gemini
- 실습 (30분)
  - 4개 툴에 모두 아래 질문 입력:  
    > 오늘 할 일을 5개 적을 테니, 우선순위와 예상 소요시간을 정해줘.
- 숙제 (10분)
  - 각 툴에 대해 “강점 2줄, 단점 2줄”을 README `month1/week1/tool-notes.md`에 기록.

#### Day 2 – 역할 맵핑

- 실습 (35분)
  - 본인 업무 10개 리스트업 (메일 작성, MMM 리포트, 캠페인 기획, 리뷰 분석, 제안서 등).
  - Perplexity에 붙여 넣고:  
    > 이 10개 업무를 Copilot / Gemini / Claude / Perplexity 중 어떤 조합으로 하면 좋을지 추천해줘.
- 숙제 (10분)
  - 추천 결과를 기반으로 “내 멀티툴 역할 지도”를 간단 표로 README `month1/week1/tool-map`에 추가.

#### Day 3 – 동일 브리프 4툴 비교 (캠페인)

- 실습 (35분)
  - 4툴에 모두 아래 브리프 입력:  
    > 역할: 한국 FMCG 마케터  
    > 목표: 여름 시즌 보디오일 디지털 캠페인 아이디어 10개  
    > 채널: 인스타그램 / 유튜브 / 카카오톡 선물하기 중심  
    > 출력: 표 형식 (아이디어 이름 / 설명 / 주요 채널 / KPI 가설)
- 숙제 (10분)
  - “가장 실무에 쓸만한 출력 1위/2위 + 이유”를 README에 5줄 이내로 기록.

#### Day 4 – 동일 브리프 4툴 비교 (리포트/데이터)

- 실습 (35분)
  - 최근 MMM/ROI 또는 리포트 요약 텍스트(1~2페이지)를 준비해서 4툴에 모두 붙여 넣고:  
    > 임원 보고용 한 장 요약 + 핵심 인사이트 5개 + 추천 액션 3개
- 숙제 (10분)
  - “숫자에 강한 툴 vs 스토리텔링 좋은 툴”을 한 줄씩 정리.

#### Day 5 – Week 1 회고

- 실습 (40분)
  - Claude에 이번 주 결과물/메모를 붙여 넣고:  
    > 각 툴에 대해, 내가 주로 어떻게 쓰면 좋을지 가이드 문서를 작성해줘.
- 숙제 (20분)
  - Claude가 만들어준 가이드 문서를 README `month1/week1/summary` 섹션으로 복붙.

### Week 2 – 프롬프트 디자인 심화

**주간 목표**

- 역할·톤·제약·포맷·체크리스트를 포함한 **대표 프롬프트 3개** 설계.

#### Day 1 – 프롬프트 예시 관찰

- 학습 (20분)
  - Perplexity at Work / 관련 영상에서 프롬프트 예시 살펴보기.
- 실습 (30분)
  - 내 업무에 맞는 프롬프트 3개 초안 작성  
    - 예: 메일, 분석 리포트, 캠페인 기획안용.
- 숙제 (10분)
  - `month1/week2/prompt-templates-draft`로 저장.

#### Day 2–4 – 하루 1템플릿 테스트

- 매일 공통
  - 10분: 전날 템플릿 복습·수정.
  - 30분: 실제 문서(오늘 써야 할 메일, 리포트, 기획안 등)에 적용.
  - 20분: 결과물과 원본 비교, 잘 먹힌 부분·이상한 부분을 5줄 안으로 기록.

#### Day 5 – 대표 프롬프트 3개 확정

- 실습 (40분)
  - Claude/Perplexity에 지금까지 템플릿과 피드백을 붙여 넣고:  
    > 나에게 최적화된 대표 프롬프트 3개를 정리해줘.
- 숙제 (20분)
  - 대표 프롬프트 3개를 README 상단 `My Prompt Library` 섹션에 고정.

**주간 목표**

- Perplexity → Claude → Copilot/Gemini로 이어지는  
  “리서치 → 구조화 → 문서/슬라이드” 플로우를 1개 주제로 완주.

#### Day 1 – 주제 선택 & 1차 리서치

- 실습 (50분)
  - 주제 선택: 예) “보디오일 카테고리 트렌드 2026 한국/글로벌”.
  - Perplexity로 리서치 후, 핵심 내용을 10~15줄로 요약.
- 숙제 (10분)
  - 요약을 README `month1/week3/topic-research`에 저장.

#### Day 2 – Claude로 구조화

- 실습 (40분)
  - Claude에 Perplexity 요약을 넣고:  
    > 1페이지 리포트 구조(배경/트렌드/경쟁사/기회/제안)와 각 섹션 요약을 작성해줘.
- 숙제 (20분)
  - Claude가 만든 구조를 README `month1/week3/report-outline`에 복붙.

#### Day 3–4 – Copilot/Gemini로 문서·슬라이드화

- 매일
  - Copilot Word/PPT 또는 Gemini Docs/Slides로  
    - Claude 구조를 실제 문서/슬라이드로 구현.
  - 전/후 스크린샷과 느낀 점을 간단히 기록.

#### Day 5 – 미니 프로젝트 정리

- 실습 (50분)
  - 최종 리포트/PPT 결과물을 `month1/week3/body-oil-trend/` 폴더에 정리.
- 숙제 (10분)
  - “이 플로우를 바로 다음 캠페인/분석에 적용한다면?” 상상 노트 5줄 작성.

### Week 4 – Claude · Perplexity로 리서치/긴 문서 정리

**주간 목표**

- 10~30페이지짜리 리포트 1개를 Claude Cowork + Perplexity로 “완전히 씹어서” 요약·인사이트 도출.

#### Day 1 – 리포트 선택 & 1차 요약

- 실습 (50분)
  - 산업/브랜드 관련 리포트 1개 업로드(또는 긴 텍스트 붙여 넣기).
  - Claude Cowork에:  
    > 이 리포트를 5개 섹션으로 나누고, 각 섹션별 3줄 요약과 핵심 수치만 추려줘.
- 숙제 (10분)
  - “이 리포트에서 내가 진짜 궁금했던 질문 3개” 기록.

#### Day 2 – Perplexity로 외부 리서치 보강

- 실습 (40분)
  - Day1 요약에서 추출한 키워드 기준으로 Perplexity에서 최신 데이터·사례 추가 리서치.
- 숙제 (20분)
  - 내부 리포트 vs Perplexity 인사이트 중 “실제 액션으로 연결 가능한 것 3개” 정리.

#### Day 3–4 – 인사이트·액션 문서화

- 매일
  - Claude에 내부 요약 + Perplexity 리서치 결과를 함께 넣고:  
    > 브랜드 마케터 입장에서 인사이트 5개와, 채널/제품/가격/프로모션 관점 액션 5개를 정리해줘.
  - 결과를 README `month1/week4/report-project`에 추가.

#### Day 5 – Month 1 회고

- 실습 (60분)
  - Claude에 1달치 노트를 넣고:  
    > 각 툴의 캐릭터(내 기준), 2달차에서 집중할 방향, 1달 동안의 성장 포인트를 정리해줘.
  - 결과를 README 상단 `My Tool Stack (v1)` 섹션에 저장.

### Week 1 – 캠페인 & 콘텐츠 (모듈 A)

**주간 목표**

- 하나의 캠페인(실제 또는 가상)을 정하고, 브리프~기획안~카피까지 end-to-end로 AI와 함께 완성.

#### Day 1 – 캠페인 브리프 정의 + 아이디어 1차 발산

- 학습 (20분)
  - ChatGPT for Marketing & Productivity with AI Tools – 카피/캠페인 챕터 개요 확인  
    - 참고용: https://thamescollege.org/course/chatgpt-marketing-productivity-ai-course/
  - Perplexity로 “FMCG 디지털 캠페인 최신 트렌드” 간단 리서치.
- 실습 (30분)
  - 실제 진행 중이거나 할 법한 캠페인 1개 브리프 작성 (브랜드, 타겟, 목표, 기간, 채널).
  - Claude에 브리프 입력:  
    > 캠페인 컨셉 5개와 각 컨셉의 핵심 메시지·메인 채널·핵심 KPI를 제안해줘.
- 숙제 (10분)
  - 괜찮은 컨셉 1~2개만 골라 README `month2/week1/campaign-brief`에 정리.

#### Day 2 – 카피 아이디어 발산 (툴별 비교)

- 실습 (40분)
  - 선택한 컨셉 1개로  
    - Gemini Docs: 인스타/유튜브/카카오 선물하기용 카피 10개 생성.  
    - Copilot Word: 동일 브리프로 카피 10개 생성.
- 숙제 (20분)
  - 두 툴 출력의 톤·구조 차이와 “좋았던 카피 3개”를 기록.

#### Day 3 – 기획서 목차·슬라이드 구조

- 학습 (15분)
  - Copilot 마케팅 모듈에서 PPT 예시 흐름 보기.
- 실습 (35분)
  - Claude에:  
    > 이 캠페인 브리프 기준으로, 임원 보고용 10~12장 슬라이드 구조와 각 슬라이드 키 메시지를 작성해줘.
  - Copilot in PowerPoint로 초안 슬라이드 생성.

#### Day 4 – 퍼널 전체 스토리 보완

- 실습 (45분)
  - Perplexity로 벤치마크 캠페인 사례 3~5개 조사.  
  - Claude에 캠페인 구조 + 벤치마크 전달 후:  
    > 퍼널 상단~하단까지 스토리, 핵심 메시지, KPI를 보완해줘.
- 숙제 (15분)
  - Before/After 차이 3개를 README에 적기.

#### Day 5 – 미니 캠페인 덱 완성

- 실습 (50분)
  - 캠페인 PPT 덱(10~12장) 완성 후 PDF로 저장.  
  - Perplexity에 덱 요약 요청 → 임원용 5줄 Summary 생성.
- 숙제 (10분)
  - “이 덱을 실제 상사에게 설명할 때 강조할 포인트 3개” 작성.

### Week 2 – 데이터 · 리서치 · 리포트 (모듈 B)

**주간 목표**

- 실제/샘플 데이터를 가지고 AI를 리서치·인사이트 파트너처럼 사용하는 감각 익히기.

  - “내가 직접 본 인사이트 vs Gemini가 준 인사이트 차이 3개” 기록.

#### Day 3 – Perplexity로 외부 맥락 보강

    > 시장/채널/타겟 관점 종합 인사이트 5개를 정리해줘.
- 숙제 (20분)
  - 종합 인사이트 5개를 README `month2/week2/insights`에 저장.

#### Day 4 – 보고서(Word/Docs) 작성

- 실습 (50분)
